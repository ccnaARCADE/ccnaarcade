# LAYER BUILDER: A BurgerTime-Inspired OSI/TCP-IP Training Game

## High Concept
**"You're a Data Packet trying to escape a malfunctioning network by building the correct protocol stacks to reach your destination. Chase down protocol components, answer questions to lock them in place, and watch your data successfully transmit - but hurry, because network errors are hot on your trail!"**

---

## Core Gameplay Loop (30-Second Cycle)

**The BurgerTime Translation:**
- **BurgerTime**: Walk over burger parts to drop them, stack ingredients, avoid enemies
- **Layer Builder**: Navigate platforms to "activate" protocol layers, answer quick questions to lock them in, build stacks from bottom-up (Physical → Application), avoid "malware baddies"

**Moment-to-Moment Gameplay:**
1. Player (Data Packet character) runs across platforms in a multi-level structure
2. Protocol layer components are scattered on platforms (like burger ingredients)
3. Walking over a component triggers a quick question (5-15 seconds to answer)
4. Correct answer = component drops/slides into the stack below
5. Wrong answer = component stays, player gets briefly stunned, enemies get closer
6. Complete all 7 layers (OSI) or 4 layers (TCP/IP) = data transmits successfully = LEVEL COMPLETE
7. Between rounds, choose upgrades/power-ups

## Mascot Design: "Packman the Data Packet"

**Character Concept:**
A friendly, energetic data packet with personality - part Pac-Man, part Q*bert charm

**Visual Description:**
- Small cubic character (16x16 pixel sprite when small, detailed art in menus)
- Made of colorful "data bits" that shimmer
- Binary code patterns flow across surface (01010101 animation)
- Two antennae on top (like WiFi signals) that droop when sad, perk up when happy
- Big expressive eyes
- Leaves a trail of sparkly "data stream" when running

**Personality/Animation:**
- **Idle**: Bounces gently, binary patterns pulse
- **Running**: Leans forward, determined face, antennae stream backward
- **Correct Answer**: Spins happily, gives thumbs up
- **Wrong Answer**: Glitches briefly, looks dizzy
- **Hit by Enemy**: Fragments into bits, reassembles with dizzy stars
- **Level Complete**: Victory dance, shoots up through the completed stack

## Enemy System: The Malware Gang

1. **"Bug Byte"** (Red worm) - Patrols horizontally, corrupts your stack if it touches you
2. **"Trojan"** (Blue box with legs) - Chases you intelligently, faster than Bug Byte
3. **"Phisher"** (Green hook character) - Steals components you've already placed
4. **"DDoS Swarm"** (Purple cluster) - Splits into 3 when you're on same platform

## Scoring & Progression

**Base Actions:**
- Correct answer (Easy): +100 points
- Correct answer (Medium): +200 points
- Correct answer (Hard): +300 points
- Layer completed: +500 points
- Full stack completed: +2000 points

**Multipliers:**
- Speed Bonus: Answer in <3 seconds = 2x points
- Streak Multiplier: Each consecutive correct answer adds +0.5x (caps at 5x)
- Flawless Layer: No wrong answers = +1000 bonus

## Game Modes

**Story Mode - "The Seven-Layer Quest":**
- 7 worlds, each representing one OSI layer
- Each world has 5 levels
- Final level = Boss Battle

**Daily Challenge:**
- One specially-designed level per day
- Global leaderboard resets daily

**Endless Mode:**
- Stacks get progressively more complex
- See how many stacks you can build before death

---

## Why This Works for Enterprise Training

- Active gameplay (not passive quiz-taking)
- Infinite replay value (high scores, daily challenges)
- Feels like a game, not a test
- Social competition via leaderboards
- 10-15 minute sessions perfect for lunch breaks
- Wrong answers teach concepts through gameplay consequences
