HOW TO PLAY

SPEED SUBNET

Quick-fire CIDR training! A host count appears - select the minimum CIDR that supports that many hosts. Build streaks for bonus points!

CLIENT SCENARIOS

Real-world network design! Each scenario is a client with specific subnet requirements. Allocate efficiently to earn 3 stars.

PACKET JOURNEY

Follow Packie the packet through the network! Learn how data travels from your computer to its destination, including encapsulation, ARP, DNS, and routing.

OSI MODEL QUIZ

Test your knowledge of the 7-layer OSI model! Learn layer names, PDUs, protocols, and how encapsulation works.

BINARY MUNCHERS

Classic arcade action! Navigate the grid and munch the correct binary/decimal values. Avoid the Glitches! Arrow keys move, SPACE for speed boost.
